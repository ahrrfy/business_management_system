# جسر الطباعة المحلي — الرؤية العربية

عميل صغير يعمل على **جهاز الكاشير/الحاسبة نفسه** ويتيح للنظام (المنشور سحابياً) الطباعة على
**أي طابعة مثبّتة على Windows** بالاسم — حرارية إيصالات، ملصقات، أو غيرها — عبر طباعة **RAW (ESC/POS)**.

## لماذا؟
الخادم السحابي لا يصل لطابعة USB محلية. هذا الجسر يسدّ الفجوة محلياً (على `127.0.0.1`) بلا اشتراك،
بلا إنترنت، وبلا تسريب بيانات لطرف ثالث. **ناقل بايتات نقيّ**: يمرّر بايتات ESC/POS التي يولّدها النظام
**حرفياً** عبر `winspool.drv` (datatype `RAW`) ⇒ **جودة وتفاصيل مطابقة تماماً** (لا تصيير، لا ضغط، لا تحجيم).

## المتطلّبات
- Windows + Node.js مثبّت (في PATH).
- لا حاجة لتعريف WinUSB (Zadig) — على عكس WebUSB. الطباعة عبر تعريف الطابعة العادي.

## التثبيت (مرّة واحدة لكل جهاز)
```powershell
# يُفضَّل تشغيله كمسؤول لضبط سياسة المتصفّح (LNA)
powershell -ExecutionPolicy Bypass -File install.ps1 -AppOrigin "https://srv1548487.hstgr.cloud" -Token "رمز-سرّي-اختياري"
```
يقوم بـ:
1. كتابة `bridge.config.json` (المنفذ/الرمز/الأصول المسموحة).
2. جدولة تشغيل الجسر **عند الدخول** (مخفي) + تشغيله الآن.
3. ضبط سياسة Chrome/Edge (`LocalNetworkAccessAllowedForUrls`) لإخفاء نافذة «الوصول للشبكة المحلية» لأصل التطبيق.

ثم في النظام: **إدارة الطابعات** ← أدخل الرمز نفسه (إن استُعمل) ← أضِف طابعة بناقل «الجسر المحلي» واخترها بالاسم ← أسنِدها لمهمة.

## التشغيل اليدوي (تطوير/اختبار)
```powershell
node bridge.mjs
```

## واجهة HTTP (محلية فقط)
| المسار | الطريقة | الوصف |
|---|---|---|
| `/health` | GET | فحص الاتصال (بلا مصادقة) |
| `/printers` | GET | تعداد كل طابعات Windows بالاسم (مصادق) |
| `/print/raw` | POST | `{printer, bytesB64, format?}` — طباعة RAW (مصادق) |

المصادقة: ترويسة `Authorization: Bearer <token>` (إن ضُبط رمز). يستمع على `127.0.0.1` فقط، ويردّ رؤوس
CORS + `Access-Control-Allow-Private-Network: true` لأصل التطبيق.

## السجلّ
يكتب الأحداث في `bridge.log` بجوار `bridge.mjs` (إقلاع + كل مهمة + كل خطأ).

## ملاحظات
- **الطابعات الشبكية** (منفذ 9100): تعمل عبر الجسر أيضاً (أو يمكن للخادم الوصول لها مباشرة).
- **ملصقات ZPL/EPL (Zebra/TSC)**: مدعومة مستقبلاً — يولّد النظام بايتات ZPL ويرسلها بـ`format:"zpl"`؛ الجسر يمرّرها حرفياً بلا تغيير.
- **مستندات A4**: تُطبع عبر **حوار المتصفّح** (vector بأعلى دقّة) لا عبر الجسر.
- استثناء Defender إن لزم: `Add-MpPreference -ExclusionPath "<مجلد الجسر>"`.
